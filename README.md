# XControllerTool
![ScreeenShot](https://i.imgur.com/uPp2X7J.jpg)

このツールはWindowsに接続されたワイヤレスXBoxコントローラについて、

* XBoxコントローラーの電源をすぐに安全に切る
* バッテリ残量を確認する

ができます。Windows専用です。

## バッテリ残量
![BatteryLevel](https://i.imgur.com/SNjBdkx.jpg)

四段階です。でもlevel0では電源が入らないから実質3段階です。

## 必須ライブラリ
* 最新のDirectX
* .Net4.5↑
